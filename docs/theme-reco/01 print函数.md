# 01 print函数

1. 语法：print()，括号内是输出的内容

2. 如果括号内是一个字符串（一串文字或者英文），必须放在单引号里面，例如 `print('helloworld') `

3. 如果括号内是一个数字，则不需要引号

4. 括号内可以出现运算

5. 运行结果会把括号内容输出在控制台

   **代码**

```python
`print('hello world')
print(666)`
```

**输出结果**

`hello world
666`
