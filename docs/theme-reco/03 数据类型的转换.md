# 03 数据类型的转换

1. 使用str()那么括号里的内容会被转换为字符串类型，整数，浮点数，布尔值都可以进行转换成字符串
2. 使用int()把括号里的内容转换为整数，只有为整数的字符串和布尔值可以转换成整数
3. 使用float()把括号里的内容转换为浮点数，只有为小数的字符串可以转换为浮点数 
4. 使用bool()把括号里的内容转换为浮点数 只有零和一可以转化为布尔值

如果想让1 + 1 = 11，那么就先把1转换为字符串

**代码**

```
print(str(1) + str(1))
```

**输出结果**

11
